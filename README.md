# Bashpack unstable

**Install**
```javascript
curl -sL https://raw.githubusercontent.com/bashpack-project/bashpack-unstable/main/bashpack.sh -o bashpack.sh \
 && chmod +x bashpack.sh \
 && sudo ./bashpack.sh -i \
 && rm bashpack.sh
```

**Uninstall**
```javascript
sudo bp --self-delete
```

**Usage**
```javascript
bp --help
```
